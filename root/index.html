  • Trainee as a Java Software Developer
    Mar 2019 - Aug 2019

    Right before my bachelor thesis my university offered me a chance to do a practical semester by a company of my choice.

    since August 2016

    Immediately after quitting my previous study I've started a new chapter. Just half a mile away from the old campus the University of Applied Sciences of Dortmund is located. Believing that this subject can provide me the best of both worlds, I've begun to start another study here.

    Study contents: Economics, Java Development, Software Construction, Web Development, Marketing, (Cost) Accounting, Project Management

    Aug 2015 - Aug 2016

I never felt like it and so I made a hard but necessary decision to quit.

    In the year I've spent there I've got taught about the Java programming language and the basics of software construction. Furthermore, I've passed a course which focused on computer structure and assembler programming.
